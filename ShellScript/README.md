# Introdução 

Essa pasta foi criada justamente para aprender shellScript em diversos terminais...
Principalmente o Bash, PowerShell e CMD... Porém irei usar mais o PowerShell.

# Como executar um script bash (extensoes .sh ou .bash) no PowerShell ou git Bash ou terminal Linux? 

use o comando 

```bash
bash code.sh

# ou 

bash code.bash
```

# Como executar scripts .ps1 no PowerShell?

**Lembrando:** Precisa fazer modificações na política de execução do seu pc
para executar um script powershell com extensão .ps1

```powershell

.\SeuScript.ps1

ou

powershell -File "path_string"

```
